# Shuvadarshan Bhual — GTM Leader & Revenue Architect

A single-file, fully static executive portfolio site. No frameworks, no build step, no external API calls — just HTML, CSS, and vanilla JavaScript, deployable by pushing one file.

**Live site:** https://therangecoder.github.io/portfolio-website/

## What this is

A portfolio documenting the GTM systems, automation, and revenue infrastructure I've led and built end-to-end — sole architect and builder on every project shown. It covers:

- **How I lead and build** — the operating principles behind the work
- **The problems I've solved** — 15 case studies across revenue operations, revenue protection, customer intelligence, finance automation, GTM strategy, and CRM architecture, each with problem / cost of inaction / solution / result / stack
- **Tools stack** — the systems I've architected around, integrated, or built on top of at the infrastructure level
- **What I believe** — 13 principles I lead and build by

Available in English and German via an instant toggle (no page reload).

## Stack

- Pure HTML / CSS / vanilla JS — no dependencies, no npm, no build tools
- Fonts: [Inter](https://fonts.google.com/specimen/Inter) + [JetBrains Mono](https://www.jetbrains.com/lp/mono/), loaded from Google Fonts
- Hosted on GitHub Pages

## Local development

No build step required. Either open `index.html` directly in a browser, or serve it locally:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.

## Deployment

This repo is deployed via **GitHub Pages** from the `main` branch. Any push to `main` updates the live site at:

https://therangecoder.github.io/portfolio-website/
